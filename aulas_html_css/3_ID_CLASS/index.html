ID e CLASS

ID

O ID funciona como um CPF. Assim como só posso ter um número de CPF, cada elemento só pode ter um valor de ID. O CPF é unico, ou seja, ninguem pode ter um CPF igual ao meu. Isso significado que o ID tambem é unico, então não é possivel repetir o mesmo valor duas vezes.

CLASS

O CLASS funciona de maneira oposta ao ID. Diferente do ID em que você, em quie só você pode ter um valor, no CLASS você pode ter varios valores. Tambem diferente do ID, em que não se pode repetir um valor, no CLASS em que você pode ultilizar o mesmo valor para varios elementos.
